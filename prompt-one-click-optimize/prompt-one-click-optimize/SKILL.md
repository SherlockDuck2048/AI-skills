# 提示词一键优化

## 任务目标

对用户提供的原始提示词进行智能优化，生成更清晰、更有效、更易执行的优化版本。

## 核心能力

- 结构优化：角色定义、任务拆解、输出格式
- 表达优化：去除歧义、明确边界、增加约束
- 效果增强：示例注入、分层指令、错误处理指导

## 优化策略

### 1. 结构检查清单

优化时按以下顺序检查并完善：

| 层级 | 要素 | 优化要点 |
|------|------|----------|
| 角色定义 | 身份/能力 | 明确AI扮演的角色，包含领域背景和能力边界 |
| 任务描述 | 输入/输出 | 清晰定义输入格式、期望输出格式 |
| 约束条件 | 禁止/限制 | 明确不做什么、边界在哪里 |
| 示例注入 |Few-shot | 提供2-3个典型输入输出示例 |
| 错误处理 | 异常情况 | 定义遇到问题时的处理方式 |

### 2. 优化操作流程

当用户说"优化"、"一键优化"或"改进这段提示词"时：

1. **接收原始提示词**：用户提供的文本
2. **分析当前结构**：识别缺少的要素
3. **应用优化策略**：逐项补充完善
4. **输出优化版本**：保持原意但更清晰有效
5. **简要说明**：指出做了哪些优化（如需要）

### 3. 优化输出格式

```
## 原始提示词
[用户提供的原文]

## 优化后提示词
[优化后的完整提示词]

## 优化说明（可选）
- 新增：[具体内容]
- 改进：[具体改动]
```

## 使用示例

### 示例1：直接优化
- **输入**：用户粘贴一段提示词，说"帮我优化一下"
- **输出**：优化后的完整提示词
- **要点**：保持原意，优先提升清晰度和可执行性

### 示例2：明确优化目标
- **输入**：用户说"帮我优化这段提示词，要更简洁"或"要更适合Claude模型"
- **输出**：符合目标要求的优化版本
- **要点**：针对用户指定的优化方向调整

### 示例3：批量优化
- **输入**：用户提供多个提示词，需要批量处理
- **输出**：逐个优化后的完整列表
- **要点**：保持风格一致性，效率优先

## 注意事项

- 不改变用户的核心意图
- 优化后提示词可直接使用
- 如用户未指定模型，按通用最佳实践优化
- 保持优化简洁，避免过度工程化
